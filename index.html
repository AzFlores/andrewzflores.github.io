Distributional Vocabulary Learning (2022)

University of Illinois at Urbana-Champaign

Project Overview

Developed a novel distributional measure (Pro-KWo) to quantify how existing vocabulary knowledge predicts the acquisition of new words in children, using over 6 million words of child-directed speech.

My Role

  • Designed and implemented the Pro-KWo metric for co-occurrence modeling.
  • Conducted statistical analyses using R (tidyverse, lme4) to validate predictive accuracy.
  • Interpreted results and drafted findings for publication in the Journal of Memory and Language.

Methods

  • Corpus Processing: Cleaned and tokenized 6M+ words of child-directed speech.
  • Type-Token Ratio: Measured lexical diversity across developmental stages.
  • Co-occurrence Modeling: Computed Proportion Known Word Co-occurrence (Pro-KWo) scores.

Key Insights

  • Words co-occurring with known vocabulary were acquired earlier (r = .62).
  • Contextual efficacy of Pro-KWo remained robust across grammatical categories.
  • Pro-KWo outperformed traditional metrics by 25% in prediction accuracy.

Applications & Impact

This work informs educational tool design by leveraging existing vocabulary knowledge to scaffold new learning and improve early interventions.

UX & Data Science Highlights

  • Data Science: Advanced corpus analytics, feature engineering, mixed-effects modeling.
  • UX Research: Insights for adaptive educational interface design.

Environmental Engagement Triggers in Minecraft (2023)

University of Illinois at Urbana-Champaign

Project Overview

Examined how Minecraft game features influence self-directed exploration and generative learning in educational environments.

My Role

  • Analyzed real-time positional data and behavioral logs from 50+ student sessions.
  • Developed automated metrics for interaction quantity and quality.
  • Evaluated relationships between feature exposure, engagement density, and learning outcomes.

Methods

  • Positional Tracking: Logged movements, tool usage, observational actions.
  • Lexical Diversity: Computed Type-Token Ratio on in-game chat and notes.
  • Spatial Aggregation: Generated heatmaps visualizing engagement intensity and quality.

Key Insights

  • Players gravitated toward central landmarks and NPC zones for deeper interactions.
  • High engagement frequency did not always correlate with high-quality observations.
  • Distinct exploration strategies emerged based on feature familiarity.

Applications & Impact

Guides game design by optimizing feature placement, aligning educational objectives, and refining engagement analytics tools.

UX & Data Science Highlights

  • Data Science: Time-series positional analysis, metric development, heatmap visualization.
  • UX Research: Behavioral insights for immersive educational interfaces.

Semantic Competition (2024)

University of Illinois at Urbana-Champaign

Project Overview

Investigated how semantically related stimuli compete during memory retrieval and affect user interactions in interfaces.

My Role

  • Designed and ran Visual World Paradigm eye-tracking experiments (n=150).
  • Integrated ResNet CNN features with Word2Vec embeddings.
  • Analyzed Bayesian mixed-effects models to quantify competition.

Methods

  • Visual World Paradigm: Tracked gaze to measure semantic activation.
  • CNN Feature Extraction: Processed images via ResNet.
  • Embedding Analysis: Calculated cosine similarity on Word2Vec vectors.

Key Insights

  • Users glance at semantically related distractors in 40% of trials.
  • Visual and linguistic contexts modulate interference.
  • Competition strength varies with task demands.

Applications & Impact

Informs interface layout and labeling to reduce cognitive interference and enhance usability.

UX & Data Science Highlights

  • Data Science: Multimodal feature integration, Bayesian modeling.
  • UX Research: Eye-tracking insights for cognitive alignment.

Similarity Judgments (2024)

University of Illinois at Urbana-Champaign

Project Overview

Explored how task instructions and context shape user perceptions of similarity between concepts, using visual and linguistic stimuli.

My Role

  • Created instruction sets for similarity rating tasks (n=200).
  • Applied CNN and Word2Vec metrics to predict judgments.
  • Performed mixed-effects regression to model feature weighting.

Methods

  • Instruction Manipulation: Conceptual vs. visual focus tasks.
  • Feature Extraction: Low/high-level visual and semantic embeddings.
  • Adaptive Weighting: Regression modeling of dynamic relevance.

Key Insights

  • Visual instructions increased reliance on CNN features by 30%.
  • Users flexibly integrate cues based on context.
  • Feature relevance shifts with task demands.

Applications & Impact

Guides recommendation engines, adaptive categorization, and context-aware UI labeling.

UX & Data Science Highlights

  • Data Science: Multimodal similarity modeling, regression analysis.
  • UX Research: Contextual perception insights for interfaces.
